# 顶部一级导航菜单样式规范

## 概述
本文档定义了顶部一级导航菜单（如"规则管理"、"综合查询"等）的严格样式规范，确保视觉一致性和用户体验。

## 样式定义

### 1. 选中态 (Active / Selected State)
**视觉特征**：当前激活页面的样式，视觉权重最高。

**样式规范**：
- **背景色 (Background)**：品牌主色高亮亮蓝色 `#1677FF`
- **形状 (Shape)**：圆角矩形容器，圆角半径 `6px`
- **文字颜色 (Text Color)**：纯白色 `#FFFFFF`，字重 `600`（加粗）
- **阴影效果**：`box-shadow: 0 4px 12px rgba(22, 119, 255, 0.3)`
- **过渡效果**：`transition: all 0.2s ease-in-out`

**对应描述**：
> "选中项呈现为亮蓝色背景色块，文字为白色，具有明显的圆角矩形轮廓，视觉权重最高。"

### 2. 悬停态 (Hover State)
**视觉特征**：鼠标经过时的反馈样式，提供弱反馈。

**样式规范**：
- **背景色 (Background)**：深灰色高亮 `#333333`
- **形状 (Shape)**：圆角矩形容器，圆角半径 `6px`（与选中态一致）
- **文字颜色 (Text Color)**：纯白色 `#FFFFFF`
- **阴影效果**：`box-shadow: 0 2px 8px rgba(0, 0, 0, 0.15)`
- **过渡效果**：`transition: all 0.2s ease-in-out`

**对应描述**：
> "鼠标悬停时，菜单项呈现为深灰色背景色块，文字保持白色，形状与选中态一致但颜色更低调。"

### 3. 默认态 (Default State)
**视觉特征**：非激活状态的常规样式。

**样式规范**：
- **背景色 (Background)**：透明
- **文字颜色 (Text Color)**：浅灰色 `#a6adb4`
- **字重**：`500`（中等）
- **字体大小**：`15px`
- **内边距**：`0 20px`

### 4. 箭头图标样式
**默认状态**：
- 颜色：`#a6adb4`
- 大小：`12px`

**选中状态**：
- 颜色：`#FFFFFF`

**悬停状态**：
- 颜色：`#FFFFFF`

## 设计原则

1. **一致性**：所有一级导航菜单项遵循相同的样式规范
2. **可访问性**：确保足够的颜色对比度（白色文字与背景色）
3. **反馈明确**：悬停和选中状态提供明确的视觉反馈
4. **性能优化**：使用 CSS 过渡效果提升用户体验
5. **响应式**：样式在不同屏幕尺寸下保持一致性

## 验证方法

1. **视觉验证**：
   - 选中态：亮蓝色背景，白色文字，圆角矩形
   - 悬停态：深灰色背景，白色文字，圆角矩形
   - 默认态：透明背景，浅灰色文字

2. **交互验证**：
   - 鼠标悬停时正确显示悬停态
   - 点击后正确显示选中态
   - 状态切换时有平滑的过渡效果

3. **代码验证**：
   - 样式代码符合规范
   - 颜色值使用预定义变量
   - 没有硬编码的样式值
